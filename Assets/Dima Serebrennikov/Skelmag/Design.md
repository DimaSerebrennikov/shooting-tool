# Skelmag

Это главный персонаж, которым управляет игрок.

### Features

У персонажа нету парметра скорости.

Передвижение воспроизводится относительно его подачи скорости на аниматор.

Поэтому увеличение скорости подразумевает изменение анимации - решается это процедурной анимацией.

### Components

Skelmag - это лишь набор данных, которые использует система Skelmag, это её сериализация.

SkelmagCore - это внутрненяя система Skelmag, которая обрабатывает более глубокие задачи.

### Systems

Состоит из таких основных систем:

- Shooting - отвечает за стрельбу, но не работает с мышкой.

- Movement - отвечает за передвижения персонажа с помощью аниматора, преобразую 2д круговой вход в 2д круговой выход.

- TargetToMouse - отвечает за расположение указателя к которому стреляет персонаж.

- MouseTrigger - выполняет дейсвите по ЛКМ, необходим для стрельбы, которое устанавливает верхний объект напрямую.
