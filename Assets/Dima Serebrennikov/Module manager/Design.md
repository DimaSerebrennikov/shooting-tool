# Module manager

Технология, которая позволяет удобно управлять модулями внутри юнити движка.

### Features

Есть главный элемент - список фильтрованных модулей.

А ниже кнопка на добавление нового компонента.

Есть такие компоненты:

- Selection component - позволяет добавлять новые модули в фильтрованный список.

- Buffer component - позволяет наполнять буффер модулями, чтобы использовать его над фильтрованным списком.

- Isolation component - отображение внутренней папки модуля.
