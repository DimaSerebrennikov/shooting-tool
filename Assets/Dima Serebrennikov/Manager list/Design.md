# Manager list

Это список менеджеров, которые используются внутри игры. Самый верхний уровень игры, под которым дальше лежит Data list.

### Manager

Менеджер - это объект, который работает с репозиторием для улучшения его интерфейса. Он может быть в виде конфигурации - когда только лишь ожидает что его используют и сам он не будет модифицироваься. Либо может быть в виде контекста - когда его используют и он при этом модифицируется.
