# Ui

Это визуальный инетрфейс с игроком.

### Features

Есть окно, которое отображается когда здоровье игрока падает ниже 0. Оно позволяет перезапустить игру.

Использует старый дизайн окна, который был разработан ранее, это окно позволяет вернуться обратно в игру или перезапустить игру, а во время его отображения игра стоит на месте.
