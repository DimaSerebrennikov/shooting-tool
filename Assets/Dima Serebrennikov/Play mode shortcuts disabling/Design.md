# Play mode shortcuts disabling

[Automatically switches editor shortcut profile when playing Unity3D · GitHub](https://gist.github.com/RodGreen/8d5f5452c459755d9bf6bad1015ef194)

Просто скопировано и вставлено. Оно решает проблему с горячими клавишами.

### Features

Для правильной работы - нужно добавить Player profile, а также удалить внутри него все shortcuts. 

Unity не предоставляет возможности автоматизировать этот момент, поэтому придётся делать вручную.
