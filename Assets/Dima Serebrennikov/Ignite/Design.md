# Ignite

Это эффект, который игрок навешивает на противников при попадании.

### Features

Каждое попадание по противнику заставляет тех гореть.

Когда игрок рядом с противником, то восстанавливает здоровье.

Имеет ссылку на здоровье игрока и на событие попадание по противнику.

Выполнено в виде системы с плохим интерфейсом.

Огонь наносит урон по противнику переодически, для этого имеет ссылку на контекст здоровья противников и по нему наносит урон.

Когда противник недоступен, то компонент игната удаляется.

Выполнено через события.

Чтобы хилить игрока, технология требует ссылку на его здоровье и местоположение.

Хил имеет дебаг сферу внутри игры.
