# Range enemy

Это противник, который стоит на месте и стреляет в игрока.

### Features

Имеет здоровье, направлен на игрока, постоянно выстреливать пули.

Когда пуля касается игрока - то наносят урон.

Когда игрок попадает по противнику - то наносит урон.

При 0 здоровья - исчезает.

Имеет физическое тело.

Пули быстрее вылетают и дальше падают с замедлением - физические с балистикой.

Выстрелы имеют расброс.

Пули имеют вид - дебажный.

- Моделька снеговика, который стреляет из небольшой пушки.

Есть лазер для дебага.

Замороженная фигурка тоже имеет вид противника. Но пока в другом материале.

Есть Vfx на пуле.

Противник более динамичен и сражается с игроком один на один достаточно уверенно.

Противник умирает от падения за пределы карты.

Пуля противника исчезает со временем.

Система пуль выполнена через СОА.
